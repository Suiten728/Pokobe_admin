# 機能
ぽこべぇ管理人Botに追加する機能は以下の通りです。</br>
💡実装済みは🔵、未実装又は協議中は🔴で分けています。</br>
## 🔵P!ping 
レイテンシーを計測します。スラッシュコマンドでも可能です。</br>
## 🔵P!lock [message_id]
そのチャンネルでそのメッセージを固定します。[message_id]はコマンド実行するチャンネル内にないとエラーが出ます。</br>
## 🔵P!eval
コード実行コマンド(is_owner)
